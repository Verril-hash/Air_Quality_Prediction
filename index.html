Dynamic Machine Learning Analysis Using PySpark

How It Works

This dashboard connects to a Flask backend that runs PySpark analysis in real-time. When you click "Run Analysis", the server executes the complete ML pipeline and streams results back to this interface dynamically.

Technology Stack

Python 3.12 PySpark 4.0 MLlib Flask HTML5 JavaScript

Pipeline Steps

  1. Load 87,672 air quality records from CSV
  2. Preprocess data and handle missing values
  3. Calculate AQI using EPA standard formula
  4. Engineer 22 features (temporal, lag, interaction)
  5. Build 6-stage ML pipeline with transformers
  6. Train Random Forest with 50 trees, maxDepth=10
  7. Evaluate model (R², RMSE, MAE)
  8. Return predictions and insights to dashboard
